# Free Fire Tournament Management Platform

Production-ready full-stack Free Fire tournament management platform built with React, TypeScript, TanStack Start, and Supabase, featuring player registration, tournament operations, match management, and administrative controls.

## Live Demo
https://freefire-tournments-ss.lovable.app

## Tech Stack
- React 19
- TypeScript
- TanStack Start
- TanStack Router
- TanStack Query
- Supabase
- Tailwind CSS
- Radix UI
- Vite

## Features
- Player Registration
- Tournament Listings
- Match Scheduling
- Tournament Operations
- Admin Management Panel
- Responsive UI
- Real-time Data Management with Supabase
